## URL
```
https://api.baitech-ipfs.net/psa/pins
```

## 描述

将文件在 Crust 网络下单，获取多个副本。提交的 Pin 请求会放入一个队列，由 Crust Pinner 按照先后顺序在 Crust 下单存储。

## 请求类型
POST

## Headers
```
"Authorization: Bearer": "<YOUR_API_KEY>"
```

## Curl 请求示例
```sh
curl -X POST 'https://api.baitech-ipfs.net/psa/pins' \
--header 'Authorization: Bearer <YOUR_API_KEY>' \
--header 'Content-Type: application/json' \
--data-raw '{
    "cid": "QmeQRHBrBdCvqx41m58w5v9wcQo5A38FzRRTv7jS4WckMV",
    "name": "README.md"
}'
```
